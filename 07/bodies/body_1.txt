    <div class="s-prose js-post-body" itemprop="text">
<p>Based on your question:</p>

<blockquote>
  <p>My question is, using the code below, how would you be able to have multiple clients connected? I've tried lists, but I just can't figure out the format for that. How can this be accomplished where multiple clients are connected at once and I am able to send a message to a specific client?</p>
</blockquote>

<p>Using the code you gave, you can do this: </p>

<pre class="lang-py s-code-block"><code class="hljs language-python"><span class="hljs-comment">#!/usr/bin/python           # This is server.py file                                                                                                                                                                           </span>

<span class="hljs-keyword">import</span> socket               <span class="hljs-comment"># Import socket module</span>
<span class="hljs-keyword">import</span> thread

<span class="hljs-keyword">def</span> <span class="hljs-title function_">on_new_client</span>(<span class="hljs-params">clientsocket,addr</span>):
    <span class="hljs-keyword">while</span> <span class="hljs-literal">True</span>:
        msg = clientsocket.recv(<span class="hljs-number">1024</span>)
        <span class="hljs-comment">#do some checks and if msg == someWeirdSignal: break:</span>
        <span class="hljs-built_in">print</span> addr, <span class="hljs-string">' &gt;&gt; '</span>, msg
        msg = raw_input(<span class="hljs-string">'SERVER &gt;&gt; '</span>)
        <span class="hljs-comment">#Maybe some code to compute the last digit of PI, play game or anything else can go here and when you are done.</span>
        clientsocket.send(msg)
    clientsocket.close()

s = socket.socket()         <span class="hljs-comment"># Create a socket object</span>
host = socket.gethostname() <span class="hljs-comment"># Get local machine name</span>
port = <span class="hljs-number">50000</span>                <span class="hljs-comment"># Reserve a port for your service.</span>

<span class="hljs-built_in">print</span> <span class="hljs-string">'Server started!'</span>
<span class="hljs-built_in">print</span> <span class="hljs-string">'Waiting for clients...'</span>

s.bind((host, port))        <span class="hljs-comment"># Bind to the port</span>
s.listen(<span class="hljs-number">5</span>)                 <span class="hljs-comment"># Now wait for client connection.</span>

<span class="hljs-built_in">print</span> <span class="hljs-string">'Got connection from'</span>, addr
<span class="hljs-keyword">while</span> <span class="hljs-literal">True</span>:
   c, addr = s.accept()     <span class="hljs-comment"># Establish connection with client.</span>
   thread.start_new_thread(on_new_client,(c,addr))
   <span class="hljs-comment">#Note it's (addr,) not (addr) because second parameter is a tuple</span>
   <span class="hljs-comment">#Edit: (c,addr)</span>
   <span class="hljs-comment">#that's how you pass arguments to functions when creating new threads using thread module.</span>
s.close()
</code></pre>

<p>As Eli Bendersky mentioned, you can use processes instead of threads, you can also check python <code>threading</code> module or other async sockets framework. Note: checks are left for you to implement how you want and this is just a basic framework.</p>

    <div class="s-prose js-post-body" itemprop="text">
<p><code>accept</code> can continuously provide new client connections. However, note that it, and other socket calls are usually blocking. Therefore you have a few options at this point:</p>

<ul>
<li>Open new threads to handle clients, while the main thread goes back to accepting new clients</li>
<li>As above but with processes, instead of threads</li>
<li>Use asynchronous socket frameworks like Twisted, or a plethora of others</li>
</ul>

    <div class="s-prose js-post-body" itemprop="text">
<p>Here is the example from the <a href="http://docs.python.org/library/socketserver.html#socketserver-tcpserver-example" rel="noreferrer">SocketServer documentation</a> which would make an excellent starting point</p>

<pre class="lang-py s-code-block"><code class="hljs language-python"><span class="hljs-keyword">import</span> SocketServer

<span class="hljs-keyword">class</span> <span class="hljs-title class_">MyTCPHandler</span>(SocketServer.BaseRequestHandler):
    <span class="hljs-string">"""
    The RequestHandler class for our server.

    It is instantiated once per connection to the server, and must
    override the handle() method to implement communication to the
    client.
    """</span>

    <span class="hljs-keyword">def</span> <span class="hljs-title function_">handle</span>(<span class="hljs-params">self</span>):
        <span class="hljs-comment"># self.request is the TCP socket connected to the client</span>
        self.data = self.request.recv(<span class="hljs-number">1024</span>).strip()
        <span class="hljs-built_in">print</span> <span class="hljs-string">"{} wrote:"</span>.<span class="hljs-built_in">format</span>(self.client_address[<span class="hljs-number">0</span>])
        <span class="hljs-built_in">print</span> self.data
        <span class="hljs-comment"># just send back the same data, but upper-cased</span>
        self.request.sendall(self.data.upper())

<span class="hljs-keyword">if</span> __name__ == <span class="hljs-string">"__main__"</span>:
    HOST, PORT = <span class="hljs-string">"localhost"</span>, <span class="hljs-number">9999</span>

    <span class="hljs-comment"># Create the server, binding to localhost on port 9999</span>
    server = SocketServer.TCPServer((HOST, PORT), MyTCPHandler)

    <span class="hljs-comment"># Activate the server; this will keep running until you</span>
    <span class="hljs-comment"># interrupt the program with Ctrl-C</span>
    server.serve_forever()
</code></pre>

<p>Try it from a terminal like this</p>

<pre class="lang-py s-code-block"><code class="hljs language-python">$ telnet localhost <span class="hljs-number">9999</span>
Trying <span class="hljs-number">127.0</span><span class="hljs-number">.0</span><span class="hljs-number">.1</span>...
Connected to localhost.
Escape character <span class="hljs-keyword">is</span> <span class="hljs-string">'^]'</span>.
Hello

$ telnet localhost <span class="hljs-number">9999</span>
Trying <span class="hljs-number">127.0</span><span class="hljs-number">.0</span><span class="hljs-number">.1</span>...
Connected to localhost.
Escape character <span class="hljs-keyword">is</span> <span class="hljs-string">'^]'</span>.
Sausage
SAUSAGEConnection closed by foreign host.
</code></pre>

<p>You'll probably need to use A <a href="http://docs.python.org/library/socketserver.html#asynchronous-mixins" rel="noreferrer">Forking or Threading Mixin</a> too</p>

<p>This is what I've  done in python 3.</p>
<pre class="lang-py s-code-block"><code class="hljs language-python"><span class="hljs-keyword">import</span> socket

<span class="hljs-keyword">from</span> threading <span class="hljs-keyword">import</span> Thread


<span class="hljs-keyword">def</span> <span class="hljs-title function_">on_new_client</span>(<span class="hljs-params">client_socket, addr</span>):
    <span class="hljs-keyword">while</span> <span class="hljs-literal">True</span>:
        data = client_socket.recv(<span class="hljs-number">1024</span>).decode(<span class="hljs-string">'utf-8'</span>)
        <span class="hljs-keyword">if</span> <span class="hljs-keyword">not</span> data:
            <span class="hljs-keyword">break</span>
        <span class="hljs-built_in">print</span>(<span class="hljs-string">f"<span class="hljs-subst">{addr}</span> &gt;&gt; <span class="hljs-subst">{data}</span>"</span>)
    client_socket.close()


<span class="hljs-keyword">def</span> <span class="hljs-title function_">main</span>():
    host = <span class="hljs-string">'0.0.0.0'</span>  <span class="hljs-comment"># allow any incoming connections</span>
    port = <span class="hljs-number">8888</span>

    s = socket.socket()
    s.bind((host, port))  <span class="hljs-comment"># bind the socket to the port and ip address</span>

    s.listen(<span class="hljs-number">5</span>)  <span class="hljs-comment"># wait for new connections</span>

    <span class="hljs-keyword">while</span> <span class="hljs-literal">True</span>:
        c, addr = s.accept()  <span class="hljs-comment"># Establish connection with client.</span>
        <span class="hljs-comment"># this returns a new socket object and the IP address of the client</span>
        <span class="hljs-built_in">print</span>(<span class="hljs-string">f"New connection from: <span class="hljs-subst">{addr}</span>"</span>)
        thread = Thread(target=on_new_client, args=(c, addr))  <span class="hljs-comment"># create the thread</span>
        thread.start()  <span class="hljs-comment"># start the thread</span>
    c.close()
    thread.join()


<span class="hljs-keyword">if</span> __name__ == <span class="hljs-string">'__main__'</span>:
    main()
</code></pre>
